📊 Employee Attrition & Retention Analysis (Power BI)

This project presents an interactive HR analytics dashboard built using Power BI to analyze employee attrition, retention trends, and workforce insights. The dashboard helps organizations identify key factors influencing employee turnover and supports data-driven HR decision-making.

The report includes detailed visualizations such as attrition by tenure, department-wise risk levels, employee distribution, and salary analysis. Dynamic filters (Department, Gender, Age, Risk Level) allow users to drill down into specific segments and uncover patterns affecting retention.

Key metrics like total employees, attrition count, average salary, and risk categorization provide a clear overview of workforce health. The dashboard also highlights high-risk groups, enabling HR teams to take proactive measures to improve employee satisfaction and reduce turnover.

This project demonstrates strong skills in data cleaning, data modeling, DAX calculations, and dashboard design, focusing on delivering meaningful business insights through visually appealing and user-friendly reports.


🔑 Key Metrics
Total Employees
Attrition Count & Attrition Rate (%)
Average Salary
Employee Tenure (Years at Company)
Department-wise Employee Distribution
Risk Level Classification (High / Medium / Low)
Attrition by Age Group & Gender
Retention Trends Over Time
Average Salary by Department
High-Risk Employee Segments


🛠️ Tech Stack
Power BI – Dashboard development & data visualization
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Power Query – Data cleaning and transformation
Microsoft Excel / CSV – Data source handling
Data Modeling – Relationships and schema design
